## Мини-сайт-поздравление

Файлы:
- `index.html` — страница «Знакомство»
- `page2.html` — лёгкая самоирония
- `page3.html` — романтичное поздравление с юмором
- `page4.html` — серьёзное поздравление со слайдером
- `styles.css` — общие стили, адаптивность, анимации
- `script.js` — плавные переходы между страницами, параллакс, слайдер

Открывать лучше с `index.html`. Все страницы расположены в корне папки.
